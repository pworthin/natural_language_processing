# PocketTranslator

**PocketTranslator** is a Python-based multilingual translation tool that leverages the Google Translate API. It supports both single and batch input, auto-detects source language, and translates into multiple target languages with robust error handling and input validation.

---

##  Features

- Translate text from a known or unknown source language
- Support for multiple destination languages at once
- Batch translation: translate a list of phrases in one go
- Skips excessive (1000+ character) phrases during batch translation
- Automatic language detection for unknown input
- Console interface using 'rich'
- Fully tested with 'pytest'

---

##  Requirements

- Python 3.8+
- See 'requirements.txt' for dependencies
